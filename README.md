## Il Progetto
Questo progetto è un **Portfolio Professionale dinamico** sviluppato per presentare le mie competenze come futuro Software Engineer. Non è solo una vetrina statica, ma un'applicazione web reattiva che integra dati in tempo reale da diverse fonti e supporta un'esperienza utente multilingue.

### Strumenti Utilizzati
Il sito è stato costruito utilizzando le più moderne tecnologie per lo sviluppo web:

- **React.js**: Core framework per la costruzione di una Single Page Application (SPA) modulare.
- **React-Bootstrap**: Per un design responsivo e una gestione layout pulita.
- **Axios**: Per il recupero asincrono di dati dalle API REST di GitHub.
- **React Context API**: Utilizzata per gestire lo stato globale della lingua (Italiano/Inglese).
- **GitHub API**: Per caricare dinamicamente i progetti da multipli profili utente.
- **GitHub Actions**: Pipeline CI/CD per l'automazione del build e del deploy.

---

## Ciclo di Vita del Progetto

Il processo di realizzazione ha seguito queste fasi principali:

1. **Pianificazione e Design**: Definizione dell'architettura delle informazioni e della struttura dei componenti (Hero, About, Projects, Skills, Contact).
2. **Sviluppo Locale**: Implementazione dei componenti con focus sulla riusabilità. Configurazione del file `config.js` per centralizzare i contenuti.
3. **Integrazione Dinamica**: Refactoring logico per permettere al portfolio di aggregare repository da **diversi account GitHub** in un'unica visualizzazione fluida.
4. **Sistema Bilingue**: Implementazione di un sistema di switch linguistico (IT/EN) che aggiorna istantaneamente tutti i testi dell'applicazione senza ricaricare la pagina.
5. **Continuous Deployment (CI/CD)**: Configurazione di workflow automatizzati con GitHub Actions per testare e pubblicare il sito su GitHub Pages ad ogni aggiornamento del codice.

---

## Funzionalità Distintive

- **Integrazione Multi-Account**: Il software interroga dinamicamente diversi profili GitHub (es. `simonkolaaa` e `SimonKolaa`) aggregando i progetti specificati.
- **Design Multilingue**: Supporto completo per italiano e inglese con gestione del contesto persistente.
- **Ottimizzazione delle Performance**: Utilizzo di skeleton per il caricamento fluido dei dati esterni.
- **Responsive Web Design**: Esperienza d'uso ottimizzata sia per desktop che per dispositivi mobile.
